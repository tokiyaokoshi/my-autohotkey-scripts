# 🚀 my-autohotkey-scripts

Windowsでの日々のPC作業、事務作業、そして開発環境を徹底的に効率化・自動化するための **AutoHotkey (v2.0)** スクリプト集です。

---

## 📖 はじめに：AutoHotkeyとは？
キーボードやマウスの操作をカスタマイズし、Windows上のあらゆる作業を自動化できる強力なツールです。
「特定のショートカットキーで独自の動作をさせる」「ワンクリックで複数のアプリを立ち上げる」といった、日々の「ちょっと面倒な反復作業」をプログラムの力で一瞬で終わらせることができます。

---

## ✨ 収録コードと解決する課題
現在、本リポジトリにはPC作業のボトルネックを解消する2つのコードが収録されています。

| 機能（スクリプト） | 解決する課題（Before） | 導入後の効果（After） |
| :--- | :--- | :--- |
| **1. クリップボード拡張**<br>(`clipboard_extensions.ahk`) | フォルダ階層を辿ってパスを手入力するのが手間。Webの文字をコピペすると、不要な色やフォントサイズまでくっついてきてしまう。 | `Ctrl+Win+C`で一瞬で絶対パスを取得。`Ctrl+Win+V`で純粋な文字データ（プレーンテキスト）のみを綺麗に貼り付け可能。 |
| **2. 作業環境の一括起動**<br>(`app_launcher.ahk`) | 毎朝、ブラウザ、VSCode、Outlook、PowerPointを一つずつクリックして開き、画面サイズを調整するのが面倒。 | スクリプトをダブルクリックするだけで、必要な全ツールが最適な画面サイズと配置で一斉に立ち上がる。 |

---

## 💡 どういう時に使うの？（具体的な利用シーン）

* **チームへの資料共有や、ドキュメント作成のとき**
エクスプローラー上のファイルを選択し、「Ctrl + Win + C」を押すだけですぐにパスをテキスト化できます。チャットツールや仕様書へスムーズに共有でき、「あのファイルどこだっけ？」というコミュニケーションコストを削減します。
* **Webサイトや別資料から情報をまとめるとき**
他人が作成したドキュメントやWebサイトの文章を引用する際、「Ctrl + Win + V」を使うことで、厄介な書式（太字、色、リンクなど）を完全にリセットし、まっさらなテキストとして流し込めます。
* **朝の業務開始・作業切り替えのとき**
「一括起動スクリプト」を実行するだけで、Gemini、Googleカレンダー、Outlook(メール)がすべて自動で立ち上がり、1秒で集中できる作業モードに入れます。

## 🚀 導入手順（ゼロから使えるようになるまで）
以下の3ステップで、誰でもすぐにこの自動化環境を構築できます。

### 1. AutoHotkey v2のインストール
パソコンにAutoHotkeyが入っていない場合は、[公式サイト](https://www.autohotkey.com/) にアクセスし、「Download v2.0」をクリックしてインストールを完了させます。

### 2. コードの書き場所（ファイルの作成）
デスクトップや任意のフォルダで右クリックし、「新規作成」＞「AutoHotkey Script」を選択します。
作成したファイル（例：`shortcuts.ahk`）を右クリックして「メモ帳」や「VSCode」などのエディタで開きます。
本リポジトリにあるお好みのコードをコピーし、真っ白なファイル内にそのまま貼り付けて上書き保存してください。

### 3. スクリプトの実行
保存した `.ahk` ファイルをダブルクリックします。
タスクバーの右下（時計の横のインジケーター）に緑色の「H」アイコンが表示されれば準備完了です。設定したショートカットキーや自動化がすぐに使えるようになります。

## ⚙️ 応用編：PC起動時にスクリプトを自動実行する
毎回手動でスクリプトを起動するのは手間なため、Windowsの「タスクスケジューラ」を使って、**PCを開いた（ログオンした）瞬間に自動で常駐させる設定**がおすすめです。
この方法なら「管理者権限」を付与できるため、タスクマネージャーなどのシステム画面上でもショートカットが正常に機能するようになります。

### 設定手順
1. Windowsのスタートメニューから **「タスク スケジューラ」** を検索して開きます。
2. 右側のメニューから **「基本タスクの作成...」** をクリックし、適当な名前（例：`AutoHotkey_Start`）を入力して「次へ」を押します。
3. **トリガー**：必ず **「ユーザーがログオンしたとき」** を選択します。（※「コンピューターの起動時」だと正常に動作しません）
4. **操作**：**「プログラムの開始」** を選択します。
5. **プログラムの指定（★重要ポイント）**：
   スクリプトファイルを直接指定するのではなく、以下のように2つに分けて入力します。
   * **プログラム/スクリプト**：AutoHotkeyの本体（例：`"C:\Program Files\AutoHotkey\v2\AutoHotkey64.exe"`）を指定します。
   * **引数の追加**：実行したいスクリプトのフルパスを `""`（ダブルクォーテーション）で囲んで入力します。（例：`"C:\Users\YourName\Desktop\clipboard_extensions.ahk"`）
6. 最後の画面で **「[完了] をクリックしたときに、このタスクのプロパティ ダイアログを開く」** にチェックを入れて完了します。
7. 開いたプロパティ画面で以下の2点を設定し、「OK」で保存します。
   * 全般タブの左下にある **「最上位の特権で実行する」** にチェックを入れる。
   * 全般タブ右下の「構成」を **「Windows 10」** に変更する。

これで設定は完了です！次回PC起動時から、自動でスクリプトが立ち上がります。
